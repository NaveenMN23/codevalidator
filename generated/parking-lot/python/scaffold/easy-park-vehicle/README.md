# Parking Lot: Park a Vehicle

## Problem Statement
The parkVehicle() function body is missing. Implement it to assign a vehicle to an available parking spot, create a ticket, and handle errors for full lot or invalid vehicle.

## Instructions
1. Explore the codebase to understand the existing structure and patterns.
2. Implement the target function — look for the `// TODO` or `# TODO` comment.
3. Handle all edge cases (not-found, forbidden, invalid state).
4. Run the tests to verify your implementation.

## How to Run
1. `pip install -r requirements.txt`
2. `uvicorn main:app --reload`
3. `pytest`
